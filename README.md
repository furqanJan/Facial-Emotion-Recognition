# Facial Emotion Recognition

## Description

A FER2013 facial emotion recognition experiment using pretrained CNN backbones and Keras image generators.

## Key Features

- FER2013 train/test workflow
- Image augmentation and normalization
- Pretrained ResNet50 feature extractor
- Seven-class emotion classifier
- Notebook-based training experiment

## Tech Stack

- Python
- Jupyter Notebook
- TensorFlow
- Keras
- ResNet50
- FER2013

## Installation

pip install tensorflow matplotlib

## Usage

Open the notebook in Kaggle or Colab, make the FER2013 dataset available at the notebook paths, and run the training cells.
